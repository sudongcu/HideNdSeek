# HideNdSeek
Simple Hide and Seek Game

## Docker
[Docker Hub](https://hub.docker.com/repository/docker/sudongcu/hidendseek/general)

**1. Pull docker image.**
```
docker pull sudongcu/hidendseek:latest
```

**2. Run docker image.**
```
docker run -d -p 5000:5000 sudongcu/hidendseek:latest
```

**3. Open website.**
```
http://127.0.0.1:5000
```

**4. Play.**

Setting
- row, col ( 5 ~ 10 )
